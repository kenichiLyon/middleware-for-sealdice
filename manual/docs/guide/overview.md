# 项目简介

middleware-for-sealdice 是在 Sealdice-core 跨机部署 OneBot V11 场景下使用的中间件。该手册旨在帮助你快速完成安装、配置与部署，并在使用过程中提供参考与故障排查。

- 常用配置与约定说明
- 与 OneBot V11 的对接要点
- 常见问题与故障排查建议

## 读者假设
