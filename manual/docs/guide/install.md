# 安装

## 获取代码
```bash
git clone <repo-url> middleware-for-sealdice
cd middleware-for-sealdice
```

## 安装依赖
```bash
pnpm install
```

## 目录结构（节选）
```text
manual/              # 手册（本目录）
  docs/
    .vitepress/
      config.mts
      theme/
        index.ts
        components/

```
